# The-Online-Carpet-Estimation-Tool-OCET-Project
Designed and developed a full-stack web application to automate and streamline the carpet estimation process for both flooring professionals and consumers. The tool allows users to input room dimensions, browse flooring products by supplier and range, and receive instant cost estimates — replacing a manual process that previously took 1–3 days.

Tech stack: 
- React.js (frontend)
- Node.js & Express.js (backend)
- MySQL (database)
- JWT authentication
- bcrypt password hashing
- WCAG 2.2 accessibility compliance. 

Key features built: 

- Relational database schema with many-to-many relationships across flooring types
- suppliers, ranges, and products
- RESTful APIs for user authentication
- flooring data retrieval, and carpet estimation logic
- Custom estimation algorithm accounting for carpet roll widths and material wastage
- Responsive, multi-device UI with user account management and saved estimations
- Requirements gathered through direct collaboration with a real flooring company (Wilton Locks Carpet Company)

Methodology: 

Waterfall lifecycle model; solo developer; delivered across a structured 21-week timeline with full documentation, risk management, and user testing phases.
